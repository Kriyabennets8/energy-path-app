# EnergyPath

EnergyPath is a Progressive Web App (PWA) that helps you discover your Ayurvedic dosha and gives guidance on yoga, breathwork, and nutrition to balance your energy.

## Features
- Dosha quiz to identify your dominant and secondary doshas
- Personalized daily wellness suggestions
- View dosha information and balancing plans
- Installable as an app on desktop or mobile

## How to use
1. Open `index.html` in a browser or visit the GitHub Pages URL
2. Take the quiz to see your dosha profile
3. View recommended practices and tips

## Technologies
- HTML, CSS, JavaScript
- PWA (manifest.json and service worker for offline support)
